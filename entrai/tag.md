# TAG

## Atributos

| Atributo  | tipo        | Descrição                               |
| --------- | ----------- | --------------------------------------- |
| key       | string      | Contém o ID da tag RFID                 |
| pessoa_id | Forigen Key | Id do usuario responsavel pela tag RFID |



## Requisições

| tipo     | path          | retorno |
| -------- | ------------- | ------- |
| **GET**  | /api/tag      |         |
| **PUT**  | /api/tag/<id> |         |
| **POST** | /api/tag      |         |
