# Sessão

## Atributos

| Atributo     | Tipo    | Descrição                            |
| ------------ | ------- | ------------------------------------ |
| idUser       | integer | Chave de usuario que criou a sessão. |
| sessionToken | string  | Token privada de sessão.             |
| publicToken  | string  | Token publica do usuario.            |
| browser      | string  | browser de acesso.                   |
